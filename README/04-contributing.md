## Contributing

We welcome new recipes, fixes to existing ones, and improvements to the tooling and the site.

The short version:

1. Read [`AGENTS.md`](./AGENTS.md) — it's the single source of truth for repository conventions
2. Read [`CONTRIBUTING.md`](./CONTRIBUTING.md) for the contribution workflow
3. Bootstrap a new cookbook with `bun run new`
4. Open a PR with green CI

The longer guide for new recipes lives in [`docs/adding-a-cookbook.md`](./docs/adding-a-cookbook.md).
